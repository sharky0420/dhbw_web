# DHBW Retro Banking Demo

Eine kleine Flask-Anwendung im klassischen Web&nbsp;1.0-Stil. Die Seite zeigt einen
Login für einen festen Demo-Account und präsentiert nach erfolgreicher Anmeldung
den Kontostand.

## Entwicklung

```bash
pip install flask
python app.py
```
